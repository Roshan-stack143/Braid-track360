# BraidTrack 360: Operator Industrial Hub

A high-end SaaS industrial monitoring platform for smart factory management, featuring real-time analytics, machine health tracking, and operator performance metrics.

## 🚀 Features

- **Real-Time Analytics:** Monitor factory operations with live data visualization.
- **Machine Health Tracking:** Track the status of individual machines (Running, Off, Issue, Service) with visual indicators.
- **Operator Performance:** View metrics and performance data for factory operators.
- **Interactive Dashboard:** A modern, glassmorphism-inspired UI with particle backgrounds and smooth animations.
- **Notifications System:** Real-time alerts for machine errors and shift reports.

## 🛠️ Tech Stack

- **Frontend Framework:** React 19 with Vite
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Charts:** Recharts
- **Language:** TypeScript

## 📦 Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/braidtrack-360.git
   cd braidtrack-360
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:3000`.
